# Contributing

Thanks for contributing to `laranail/license-verifier-ui`.

- Discuss substantial changes in an issue first.
- `composer format` (Pint) for style; `composer test` (Pest) must stay green.
- Each preset lives under `src/<Preset>/` as a self-contained module (PHP plus
  its co-located `config/`, `routes/` and `resources/`). Keep that boundary.
- Commits are authored under your own git identity.
