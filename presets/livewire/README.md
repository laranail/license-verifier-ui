> Livewire UI preset for [`laranail/license-verifier-ui`](https://opensource.simtabi.com/documentation/laranail/license-verifier-ui/) — the generator writes owned `lv-activation-form` and `lv-status-widget` Livewire components (extending this package) into your app. Themes: tailwind, bootstrap, unstyled, custom.

Requires PHP `^8.4 || ^8.5` on Laravel `^13` with `livewire/livewire ^3.5`; pulls `laranail/license-verifier-ui` (the core) automatically.

## Install

```bash
composer require laranail/license-verifier-ui-livewire
php artisan laranail::license-verifier-ui.install livewire
```
