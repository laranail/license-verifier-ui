# laranail/license-verifier-ui-livewire

[![License: MIT](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)

> Livewire UI preset for [`laranail/license-verifier-ui`](https://opensource.simtabi.com/license-verifier-ui/).

```bash
composer require laranail/license-verifier-ui-livewire
php artisan laranail::license-verifier-ui.install livewire
```

Generates owned `lv-activation-form` and `lv-status-widget` Livewire components (extending this
package) into your app. Themes: **tailwind, bootstrap, unstyled, custom**.

Requires PHP `^8.4 || ^8.5`, Laravel `^13`, `livewire/livewire ^3.5`, and
`laranail/license-verifier-ui`. See the [ecosystem docs](https://opensource.simtabi.com/license-verifier-ui/docs/).
